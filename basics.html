Basic elements of the C ++ language

The C ++ language was invented by Bjarne Stroustrup in 1979, as an extension of the C language. The C language was invented in 1969-1973 by Dennis Ritchie to make the Unix operating system. Thus, almost all programs written in C can be compiled in C ++, possibly with very few modifications.

Programming languages ​​are languages ​​similar to human language. They contain words (quite a few), punctuation marks, mathematical operations and have writing rules. Programs running on any computer have been written in a programming language. There are many programming languages, such as C, C ++, Pascal, Java, Python, PHP, Javascript, etc.

A program written in a programming language is called a source program and must be translated into a language that the processor understands, called machine code, or executable program. For some programming languages ​​the translation operation is called compilation (the case of C, C ++, Pascal, etc.), for other languages ​​(PHP, Python, Javascript, etc.) the translation operation is called interpretation. The translation is done by a specialized program called a compiler or interpreter.
